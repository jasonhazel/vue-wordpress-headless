# VueJS + WordPress Headless



## quickstart
- Install [Docker](https://docs.docker.com/get-docker/)
- Clone repository
- `docker-compose up`

- Clean WordPress installation at [localhost:8000](http://localhost:8000)

- VueJS Frontend POC at [localhost:8080](http://localhost:8080)
